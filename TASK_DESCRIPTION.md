# Task: Target Brazil E-Commerce Business Case Analysis

Course: Scaler Academy Data Science & ML Course  
Module: SQL

Objective  
Analyze Target Brazil's 100,000+ orders (2016-2018) using BigQuery SQL to identify business insights and provide strategic recommendations.

## Analysis Sections (20 SQL Queries)

| Section | Queries | Focus |
| :---- | :---- | :---- |
| Initial Exploration | 3 | Data structure, time range, geographic coverage |
| In-Depth Exploration | 3 | Trends, seasonality, order patterns |
| E-Commerce Evolution | 2 | Growth by state, customer distribution |
| Impact on Economy | 3 | Revenue growth, pricing, freight costs |
| Sales, Freight & Delivery | 4 | Delivery times, regional analysis, SLAs |
| Payment Analysis | 2 | Payment types, installments, trends |
| Additional Insights | 3 | Status, CLV, sellers, products |

## Deliverables

1 PDF Report containing:

- Executive summary  
- All SQL queries with results  
- Key insights for each section  
- Strategic recommendations

## Technical Requirements

Skills Demonstrated:

- Advanced SQL (CTEs, window functions, joins)  
- BigQuery (100K+ record dataset)  
- Data exploration & trend analysis  
- Business metrics quantification  
- Strategic recommendations

Database: Google BigQuery (Standard SQL)  
Tables: 8 (orders, customers, order\_items, products, payments, reviews, sellers, product\_category\_translation)  
Records: 100,000+ orders | 300,000+ items | 3,000+ sellers

## Learning Outcomes

After completing this task:

- Master BigQuery SQL for large datasets  
- Use CTEs & window functions effectively  
- Translate data into business insights  
- Develop strategic recommendations with ROI  
- Build professional analytics reports  
- Communicate findings to non-technical stakeholders
